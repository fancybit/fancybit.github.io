htop详解

<p style="margin: 10px auto; padding: 0px;">
  This is htop, an interactive process viewer for Linux. It is a text-mode application (for console or X terminals) and requires ncurses.
</p>

<p style="margin: 10px auto; padding: 0px;">
  <strong style="margin: 0px; padding: 0px;">Comparison between htop and top</strong>
</p>

<ul style="margin-left: 30px; padding: 0px; word-break: break-all;" class=" list-paddingleft-2">
  <li>
    <p>
      In 'htop' you can scroll the list vertically and horizontally to see all processes and complete command lines.
    </p>
  </li>
  
  <li>
    <p>
      In 'top' you are subject to a delay for each unassigned key you press (especially annoying when multi-key escape sequences are triggered by accident).
    </p>
  </li>
  
  <li>
    <p>
      'htop' starts faster ('top' seems to collect data for a while before displaying anything).
    </p>
  </li>
  
  <li>
    <p>
      In 'htop' you don't need to type the process number to kill a process, in 'top' you do.
    </p>
  </li>
  
  <li>
    <p>
      In 'htop' you don't need to type the process number or the priority value to renice a process, in 'top' you do.
    </p>
  </li>
  
  <li>
    <p>
      'htop' supports mouse operation, 'top' doesn't
    </p>
  </li>
  
  <li>
    <p>
      'top' is older, hence, more used and tested.
    </p>
  </li>
</ul>

<p style="margin: 10px auto; padding: 0px;">
  htop 是Linux系统中的一个互动的进程查看器，一个文本模式的应用程序(在控制台或者X终端中)，需要ncurses。
</p>

<p style="margin: 10px auto; padding: 0px;">
  与Linux传统的top相比，htop更加人性化。它可让用户交互式操作，支持颜色主题，可横向或纵向滚动浏览进程列表，并支持鼠标操作。
</p>

<p style="margin: 10px auto; padding: 0px;">
  与top相比，htop有以下优点：
</p>

<ul style="margin-left: 30px; padding: 0px; word-break: break-all;" class=" list-paddingleft-2">
  <li>
    <p>
      可以横向或纵向滚动浏览进程列表，以便看到所有的进程和完整的命令行。
    </p>
  </li>
  
  <li>
    <p>
      在启动上，比top 更快。
    </p>
  </li>
  
  <li>
    <p>
      杀进程时不需要输入进程号。
    </p>
  </li>
  
  <li>
    <p>
      htop 支持鼠标操作。
    </p>
  </li>
  
  <li>
    <p>
      top 已经很老了。
    </p>
  </li>
</ul>

<p style="margin: 10px auto; padding: 0px;">
  htop 官网：<a title="http://htop.sourceforge.net/" href="http://htop.sourceforge.net/" style="margin: 0px; padding: 0px; color: rgb(100, 102, 179);" target="_blank"  rel="nofollow" >http://htop.sourceforge.net/</a>
</p>

        <strong style="margin: 0px; padding: 0px;">a (在有多处理器的机器上)</strong>&nbsp; &nbsp; 设置 CPU affinity: 标记一个进程允许使用哪些CPU
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        <strong style="margin: 0px; padding: 0px;">u</strong>&nbsp; &nbsp; 显示特定用户进程
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        <strong style="margin: 0px; padding: 0px;">M&nbsp;</strong>&nbsp;&nbsp; 按Memory 使用排序
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        <strong style="margin: 0px; padding: 0px;">P</strong>&nbsp;&nbsp;&nbsp; 按CPU 使用排序
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        <strong style="margin: 0px; padding: 0px;">T&nbsp;</strong>&nbsp;&nbsp; 按Time+ 使用排序
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        <strong style="margin: 0px; padding: 0px;">F</strong>&nbsp;&nbsp;&nbsp; 跟踪进程: 如果排序顺序引起选定的进程在列表上到处移动，让选定条跟随该进程。这对监视一个进程非常有用：通过这种方式，你可以让一个进程在屏幕上一直可见。使用方向键会停止该功能。
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        <strong style="margin: 0px; padding: 0px;">K</strong>&nbsp;&nbsp;&nbsp; 显示/隐藏内核线程
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        <strong style="margin: 0px; padding: 0px;">H&nbsp;</strong>&nbsp;&nbsp; 显示/隐藏用户线程
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        <strong style="margin: 0px; padding: 0px;">Ctrl-L</strong>&nbsp; &nbsp; 刷新
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        <strong style="margin: 0px; padding: 0px;">Numbers&nbsp;</strong>&nbsp;&nbsp; PID 查找: 输入PID，光标将移动到相应的进程上
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        <strong style="margin: 0px; padding: 0px;">四、htop 使用</strong>
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        4.1. 显示自带帮助
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        鼠标点击Help或者按F1 显示自带帮助
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        <a href="https://images0.cnblogs.com/blog/370046/201301/12224049-575f66ab3d964c5cb419959aee9032b1.jpg" style="margin: 0px; padding: 0px; color: rgb(100, 102, 179);" target="_blank"  rel="nofollow" ><img loading="lazy" src="https://images0.cnblogs.com/blog/370046/201301/12224050-75be8605449248f999bde75c6697522f.jpg" title="htop详解  htop linux 运维 第4张" alt="htop详解  htop linux 运维 第4张" width="900" height="465" border="0" style="margin: 0px; padding: 0px; border: 0px; max-width: 900px; height: auto;" /></a>
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        4.2. htop 设定
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        鼠标点击Setup或者按下F2 之后进入htop 设定的页面，Meters 页面设定了顶端的一些信息显示，顶端的显示又分为左右两侧，到底能显示些什么可以在最右侧那栏新增，要新增到上方左侧（F5）或是右侧（F6）都可以，这就是个人设定的范围了。这里多加了一个时钟。
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        <a href="https://images0.cnblogs.com/blog/370046/201301/12224051-9b4d728776d1499190c9d21ab3615f5a.jpg" style="margin: 0px; padding: 0px; color: rgb(100, 102, 179);" target="_blank"  rel="nofollow" ><img loading="lazy" src="https://images0.cnblogs.com/blog/370046/201301/12224052-570be85130854977aa4bc9f796fb0eb1.jpg" title="htop详解  htop linux 运维 第5张" alt="htop详解  htop linux 运维 第5张" width="900" height="465" border="0" style="margin: 0px; padding: 0px; border: 0px; max-width: 900px; height: auto;" /></a>
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        上方左右两栏的显示方式分为Text Bar Graph Led 四种，下图我就把 cpu memory swap 改成文本模式显示，然后右栏的改成Bar 显示，clock 用LED方式显示。数据显示都差不多，只是这样看有点不习惯了。
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        <a href="https://images0.cnblogs.com/blog/370046/201301/12224052-befc1e283e4c4576a59f89e6f72a4337.jpg" style="margin: 0px; padding: 0px; color: rgb(100, 102, 179);" target="_blank"  rel="nofollow" ><img loading="lazy" src="https://images0.cnblogs.com/blog/370046/201301/12224053-eca0d56c388240e495c0b3226ed45472.jpg" title="htop详解  htop linux 运维 第6张" alt="htop详解  htop linux 运维 第6张" width="900" height="465" border="0" style="margin: 0px; padding: 0px; border: 0px; max-width: 900px; height: auto;" /></a>
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        关于Display options 的设定，可要根据管理者自己的需要来设定。
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        <a href="https://images0.cnblogs.com/blog/370046/201301/12224054-b8d7722ac20b421299bd0c72b3394198.jpg" style="margin: 0px; padding: 0px; color: rgb(100, 102, 179);" target="_blank"  rel="nofollow" ><img loading="lazy" src="https://images0.cnblogs.com/blog/370046/201301/12224054-97538b88011d4b6fb49e7d48ec7b4263.jpg" title="htop详解  htop linux 运维 第7张" alt="htop详解  htop linux 运维 第7张" width="900" height="465" border="0" style="margin: 0px; padding: 0px; border: 0px; max-width: 900px; height: auto;" /></a>
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        颜色选择，除了基本的颜色显示之外，htop 还提供了换面板的功能，其实也只是改变一些色彩显示的设定，虽然说不能自定义到细部的颜色显示，但是至少提供了几种风格可以选择。
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        <a href="https://images0.cnblogs.com/blog/370046/201301/12224055-b6a13eb2abaa45fab167e61a8db12356.jpg" style="margin: 0px; padding: 0px; color: rgb(100, 102, 179);" target="_blank"  rel="nofollow" ><img loading="lazy" src="https://images0.cnblogs.com/blog/370046/201301/12224056-2f6114304f37492095545a4ad6a68986.jpg" title="htop详解  htop linux 运维 第8张" alt="htop详解  htop linux 运维 第8张" width="900" height="465" border="0" style="margin: 0px; padding: 0px; border: 0px; max-width: 900px; height: auto;" /></a>
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        最后一项的设定是调整 Columns 的显示，就是在一般htop 指令进来希望可以看到的什么样的数据及信息，字段的调整可以在这边做个人化的设定，一般使用系统默认值就好了。
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        <a href="https://images0.cnblogs.com/blog/370046/201301/12224058-1ef4f726ccd64dfbaab68142dc9a95be.jpg" style="margin: 0px; padding: 0px; color: rgb(10

0, 102, 179);" target="_blank" rel=“nofollow” > 

      <p style="margin: 10px auto; padding: 0px;">
        4.3. 搜索进程
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        鼠标点击Search 或者按下F3 或者输入"/"， 输入进程名进行搜索，例如搜索ssh
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        <a href="https://images0.cnblogs.com/blog/370046/201301/12224100-dab0040192b24614b44d1e28d1b4badb.jpg" style="margin: 0px; padding: 0px; color: rgb(100, 102, 179);" target="_blank"  rel="nofollow" ><img loading="lazy" src="https://images0.cnblogs.com/blog/370046/201301/12224101-da92613d580148e7a74b5bde960bbbb8.jpg" title="htop详解  htop linux 运维 第10张" alt="htop详解  htop linux 运维 第10张" width="900" height="465" border="0" style="margin: 0px; padding: 0px; border: 0px; max-width: 900px; height: auto;" /></a>
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        4.4. 过滤器
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        按下F4，进入过滤器，相当于关键字搜索，不区分大小写，例如过滤dev
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        <a href="https://images0.cnblogs.com/blog/370046/201301/12224102-6339cac4da0b482097e1a5b218b523c4.jpg" style="margin: 0px; padding: 0px; color: rgb(100, 102, 179);" target="_blank"  rel="nofollow" ><img loading="lazy" src="https://images0.cnblogs.com/blog/370046/201301/12224103-ee01abbe5a7d4f5388057a52bd52e109.jpg" title="htop详解  htop linux 运维 第11张" alt="htop详解  htop linux 运维 第11张" width="900" height="465" border="0" style="margin: 0px; padding: 0px; border: 0px; max-width: 900px; height: auto;" /></a>
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        4.5. 显示树形结构
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        输入"t"或按下F5，显示树形结构，意思跟pstree 差不多，能看到所有程序树状执行的结构，这对于系统管理来说相当方便，理清程序是如何产生的，当然树状结构的浏览也可以依照其他数据来排序。
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        <a href="https://images0.cnblogs.com/blog/370046/201301/12224104-f5d4a8c905cd4a83b262ad2e5944f3fe.jpg" style="margin: 0px; padding: 0px; color: rgb(100, 102, 179);" target="_blank"  rel="nofollow" ><img loading="lazy" src="https://images0.cnblogs.com/blog/370046/201301/12224105-44b98e747b494234929f8af4e3eeb5e4.jpg" title="htop详解  htop linux 运维 第12张" alt="htop详解  htop linux 运维 第12张" width="900" height="465" border="0" style="margin: 0px; padding: 0px; border: 0px; max-width: 900px; height: auto;" /></a>
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        4.6. 选择排序方式
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        按下F6 就可以选择依照什么来排序，最常排序的内容就是cpu 和memory 吧！
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        <a href="https://images0.cnblogs.com/blog/370046/201301/12224107-6a091d2523cd4829b243f635b1c32a0a.jpg" style="margin: 0px; padding: 0px; color: rgb(100, 102, 179);" target="_blank"  rel="nofollow" ><img loading="lazy" src="https://images0.cnblogs.com/blog/370046/201301/12224111-2f34d96612bd420ebcf9471c7339fc63.jpg" title="htop详解  htop linux 运维 第13张" alt="htop详解  htop linux 运维 第13张" width="900" height="465" border="0" style="margin: 0px; padding: 0px; border: 0px; max-width: 900px; height: auto;" /></a>
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        4.7 操作进程
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        F7、F8分别对应nice-和nice+，F9对应kill给进程发信号，选好信号回车就OK了
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        <a href="https://images0.cnblogs.com/blog/370046/201301/12224112-9dc1706a8dbd4972b78a78af34f75ec7.jpg" style="margin: 0px; padding: 0px; color: rgb(100, 102, 179);" target="_blank"  rel="nofollow" ><img loading="lazy" src="https://images0.cnblogs.com/blog/370046/201301/12224113-0e383c20b0e74196a393586b72a67ba1.jpg" title="htop详解  htop linux 运维 第14张" alt="htop详解  htop linux 运维 第14张" width="900" height="465" border="0" style="margin: 0px; padding: 0px; border: 0px; max-width: 900px; height: auto;" /></a>
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        4.8. 显示某个用户的进程，在左侧选择用户
      </p>
      
      <p style="margin: 10px auto; padding: 0px;">
        输入"u"，在左侧选择用户
      </p>
