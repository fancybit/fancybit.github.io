VSIX项目，添加右键菜单（二）

<p style="margin: 10px auto;">
  项目上右击，添加新建项
</p>

<p style="margin: 10px auto;">
  <img src="https://images2017.cnblogs.com/blog/705223/201708/705223-20170807172213034-2125231860.png" title="VSIX项目，添加右键菜单（二）  visual studio vsix 第1张" alt="VSIX项目，添加右键菜单（二）  visual studio vsix 第1张" style="border: 0px; max-width: 100%; height: auto;" />
</p>

<p style="margin: 10px auto;">
  随便起一个名字，例如我的是&nbsp; RightCmd
</p>

<p style="margin: 10px auto;">
  <img src="https://images2017.cnblogs.com/blog/705223/201708/705223-20170807172300112-1153425242.png" title="VSIX项目，添加右键菜单（二）  visual studio vsix 第2张" alt="VSIX项目，添加右键菜单（二）  visual studio vsix 第2张" style="border: 0px; max-width: 100%; height: auto;" />
</p>

<p style="margin: 10px auto;">
  2、里边有一个vsct的文件，这个文件相当于MVC里的html，是用于配置显示的。
</p>

<p style="margin: 10px auto;">
  打开它，里边前几行，有一个引入的文件，vsshlids.h，这个文件，你可以去&nbsp; &nbsp; &nbsp; 项目路径\packages\Microsoft.VSSDK.BuildTools.14.3.25407\tools\vssdk\inc&nbsp; &nbsp;（注意版本号）下找，里边是vs自带的命令
</p>

<p style="margin: 10px auto;">
  其中比较关键的几个
</p>

<p style="margin: 10px auto;">
  guidSHLMainMenu&nbsp; 是vs主窗体用的（不知道对不对，但意思就是说vs的主要窗体）。
</p>

<p style="margin: 10px auto;">
  IDM_VS_CTXT_SOLNNODE&nbsp; 是指的解决方案资源管理器里的&nbsp; &nbsp; 解决方案，
</p>

<p style="margin: 10px auto;">
  IDM_VS_CTXT_SOLNFOLDER&nbsp; &nbsp;&nbsp;是指的解决方案资源管理器里的&nbsp; &nbsp; 解决方案里的文件夹，不是项目里的哈，这个文件夹是虚拟的，没有实际的文件夹映射，
</p>

<p style="margin: 10px auto;">
  IDM_VS_CTXT_PROJNODE&nbsp;&nbsp;是指的解决方案资源管理器里的&nbsp; &nbsp;项目，
</p>

<p style="margin: 10px auto;">
  IDM_VS_CTXT_FOLDERNODE&nbsp;&nbsp;是指的解决方案资源管理器里的&nbsp; &nbsp;项目里的文件夹，
</p>

<p style="margin: 10px auto;">
  IDM_VS_CTXT_ITEMNODE&nbsp; 是指的解决方案资源管理器里的&nbsp; &nbsp;项目里的项，就例如cs、js文件
</p>

<p style="margin: 10px auto;">
  其他的暂时不是特别清楚
</p>

<p style="margin: 10px auto;">
  3、这个vsct文件呢，和平常的定义界面的逻辑一样，你得先定义
</p>

<p style="margin: 10px auto;">
  &nbsp; &nbsp;<Symbols>&nbsp; 里边定义一个&nbsp; &nbsp;GuidSymbol，name随便起一个（guidItemNodeCmdSet），value 是一个guid，你可以使用vs自带的guid生成工具
</p>

<p style="margin: 10px auto;">
  <img src="https://images2017.cnblogs.com/blog/705223/201708/705223-20170807173634221-487392161.png" title="VSIX项目，添加右键菜单（二）  visual studio vsix 第3张" alt="VSIX项目，添加右键菜单（二）  visual studio vsix 第3张" style="border: 0px; max-width: 100%; height: auto;" />
</p>

<p style="margin: 10px auto;">
  &nbsp;
</p>

<p style="margin: 10px auto;">
  &nbsp;<img src="https://images2017.cnblogs.com/blog/705223/201708/705223-20170807173657643-1726593481.png" title="VSIX项目，添加右键菜单（二）  visual studio vsix 第4张" alt="VSIX项目，添加右键菜单（二）  visual studio vsix 第4张" style="border: 0px; max-width: 100%; height: auto;" />
</p>

<p style="margin: 10px auto;">
  选择注册表格式，点“复制”，把他放到你的GuidSymbol 的&nbsp; value 里
</p>

<p style="margin: 10px auto;">
  <img src="https://images2017.cnblogs.com/blog/705223/201708/705223-20170807173800674-432150647.png" title="VSIX项目，添加右键菜单（二）  visual studio vsix 第5张" alt="VSIX项目，添加右键菜单（二）  visual studio vsix 第5张" style="border: 0px; max-width: 100%; height: auto;" />
</p>

<p style="margin: 10px auto;">
  这就相当于&nbsp; 定义了一个变量容器，变量容器名 叫&nbsp;&nbsp;guidItemNodeCmdSet，对映的一个绝对名字&nbsp; ，就是那串GUID
</p>

<p style="margin: 10px auto;">
  里边定义了一个变量，名 叫 ItemNode, 相对名字是&nbsp; 0x1004（这个随便，但是在同一个变量容器里，这个值不能一样）
</p>

<p style="margin: 10px auto;">
  &nbsp;4、下边，定义一个Group，相当于一个容器
</p>

<p style="margin: 10px auto;">
  在Groups 里边加上这个
</p>

<p style="margin: 10px auto;">
  &nbsp;
</p>

<div class="cnblogs_code" style="background-color: rgb(245, 245, 245); border: 1px solid rgb(204, 204, 204); padding: 5px; overflow: auto; margin: 0px; color: rgb(0, 0, 0); font-family: &quot;Courier New&quot; !important; font-size: 12px !important;">
  <pre style="margin-top: 0px; margin-bottom: 0px; white-space: pre-wrap; overflow-wrap: break-word; font-family: &quot;Courier New&quot; !important;">&lt;!--&nbsp;项&nbsp;--&gt;&lt;Group&nbsp;guid="guidItemNodeCmdSet"&nbsp;id="ItemNode"&nbsp;priority="0x0100"&gt;

    <Parent guid=“guidSHLMainMenu” id=“IDM_VS_CTXT_ITEMNODE”/></Group> 

<p style="margin: 10px auto;">
  &nbsp;
</p>

<p style="margin: 10px auto;">
  　其中，Group的name&nbsp; 就是上边的那个变量容器，ID 就是上边的变量
</p>

<p style="margin: 10px auto;">
  注意的是&nbsp; Parent，guid写的是guidSHLMainMenu&nbsp; ，id 是&nbsp;&nbsp;IDM_VS_CTXT_ITEMNODE（是指的解决方案资源管理器里的&nbsp; &nbsp;项目里的项，就例如cs、js文件），这表示，如果产生效果，那就是在解决方案里的项上产生
</p>

<p style="margin: 10px auto;">
  此示例，就是指的在&nbsp; 项上右击，在右击菜单里新增一个按钮
</p>

<p style="margin: 10px auto;">
  如果你想把按钮放到项目上、文件夹上等等，那就改变这个值，取值在本文（2）里边。如果想同时在多个上边都有按钮，那就重复 步骤3、4，把id改一下
</p>

<p style="margin: 10px auto;">
  5、按钮
</p>

<p style="margin: 10px auto;">
  在 Buttons 节点下，新增一个按钮（自动生成的那个也行）
</p>

<p style="margin: 10px auto;">
  注意，Parent里的guid 和id&nbsp; 改成&nbsp; group的guid&nbsp; 和id
</p>

  6、运行，按一下F5,启动&nbsp; vs的运行实例，然后随便打开一个项目
</p>
